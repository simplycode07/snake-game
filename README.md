
# Snake Game CLI
Simple snake game for commandline made with c
use WASD to move

### How to Run -

```bash
$ git clone https://github.com/simplycode07/snake-game.git
$ cd ./snake-game
```

For linux:
```bash 
$ gcc main.c -o main.out && ./main.out
```

For windows:
```bash
$ gcc main.c -o main.exe && ./main.exe
```
